PyTorch Tensor Basics

Overview

This notebook introduces the basics of creating and working with tensors in PyTorch. It covers different ways to create tensors, understand data types, and move tensors between CPU and GPU.

Topics Covered
	•	Creating tensors in multiple way
	•	Random tensor generation
	•	Tensor data types
	•	Creating tensors like existing tensors
	•	Moving tensors between CPU and GPU

Requirements
	•	Python 3.8+
	•	PyTorch
	•	Jupyter Notebook or Jupyter Lab

Usage

Open the notebook in Jupyter and run the cells in order to explore PyTorch tensor fundamentals.
